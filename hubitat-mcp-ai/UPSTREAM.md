# Upstream attribution

This add-on is designed around:

- Project: `kingpanther13/Hubitat-local-MCP-server`
- Repository: https://github.com/kingpanther13/Hubitat-local-MCP-server
- Upstream package name: MCP Rule Server
- Licence: MIT

The Hubitat application, MCP tool catalogue, rule engine, safety model, device access gates, and tool guidance remain the work of the upstream project and its contributors.

This companion add-on supplies a local Ollama MCP client, deterministic offline fallback, and a HomeBrain-style browser interface. It does not replace the upstream Hubitat application.
