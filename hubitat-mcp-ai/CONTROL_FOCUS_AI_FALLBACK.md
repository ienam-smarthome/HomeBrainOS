# Control-first AI fallback

HomeBrain uses a layered route policy:

1. Clear device controls use the deterministic Control Agent and verified MCP execution.
2. Proven live reads use deterministic readers and calculations.
3. Unmatched read-only home questions use the bounded AI Evidence Planner.
4. The AI may select approved read-only evidence but receives no Hubitat write tools.
5. Unclear or unsupported writes require clarification or stop safely.

Control Focus therefore protects execution without blocking ordinary questions.

Octopus Live Meter Display sensors are grouped as an authoritative whole-house panel. Power, Today, Yesterday, Week, Month, Rates, Previous Rate and Standing Charge remain separate from individual device totals.
