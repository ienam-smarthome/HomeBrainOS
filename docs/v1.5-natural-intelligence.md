# HomeBrain OS v1.5 Natural Intelligence

Sprint 1 implementation started on `feature/v1.5-natural-intelligence`.

Implemented so far:

- Natural response formatter design for watts, kilowatts, kilowatt-hours, and GBP.
- Energy Advisor target wording: today/yesterday comparison, largest current consumers, and one practical recommendation.
- Daily Briefing target wording: occupancy, lights on, energy today, alerts, and recommendation.
- Home Health target wording: score, deductions, stale devices, and recommendation.

Note: the first code pass adds the extension in `homebrainos/rootfs/app/sitecustomize.py` so it can be reviewed separately before merging deeper into `main.py`.
