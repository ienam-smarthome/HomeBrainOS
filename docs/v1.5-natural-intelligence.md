# HomeBrain OS v1.5 Natural Intelligence

Sprint 1 implementation lives on `feature/v1.5-natural-intelligence`.

Implemented:

- Natural response formatting for watts, kilowatts, kilowatt-hours, and GBP.
- Energy Advisor wording with today/yesterday comparison, largest current consumers, and one practical recommendation.
- Daily Briefing wording with occupancy, lights on, energy today, alerts, and recommendation.
- Home Health wording with score, deductions, stale devices, and recommendation.
- Stable v1.5 API aliases:
  - `/api/home-context`
  - `/api/briefing`
  - `/api/home-health-score`
- Assistant handling for questions such as `how much have I spent today?` and `anything unusual?`.

Implementation note:

The first code pass adds the extension in `homebrainos/rootfs/app/sitecustomize.py` so Sprint 1 can be reviewed separately before merging deeper into the large `main.py` file.
