# Install on Home Assistant OS

1. Copy `addon/homebrainos` to `/addons/homebrainos` using Samba Share.
2. In Home Assistant, go to Settings → Add-ons → Add-on Store → ⋮ → Reload.
3. Open Local add-ons → HomeBrain OS.
4. Configure Hubitat Maker API settings.
5. Install and start the add-on.
6. Open the Web UI.
