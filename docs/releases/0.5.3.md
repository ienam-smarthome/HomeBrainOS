# HomeBrain 0.5.3 — Control Agent AI rescue

Exact controls remain deterministic and fast. When a deterministic control intent cannot resolve safely, HomeBrain now performs one local, tool-free AI reinterpretation using the same strict `ControlIntent` schema.

The rescued plan is accepted only when Python resolves it more safely than the original plan. AI never receives MCP command tools or chooses device IDs. If rescue is no better, HomeBrain preserves the original clarification and sends no command.

The Control Agent graph now excludes sensor-only devices such as Lux, illuminance, motion, presence, battery, temperature and humidity sensors from control choices while retaining clear actuators whose compact state is temporarily missing.
