# HomeBrain 0.6.0 — Agent-First Control

HomeBrain now follows the successful Claude MCP interaction pattern for device controls:

1. Understand the natural instruction before selecting a read or control route.
2. Supply the selected-device inventory to a strict structured interpreter.
3. Produce a tool-free `ControlIntent`; models cannot choose Hubitat IDs or execute commands.
4. Resolve devices and capabilities deterministically in Python.
5. Apply confirmation and risk policy before any write.
6. Execute through Hubitat MCP and verify the final state.

Exact controls remain deterministic and AI-free. Natural controls use local Qwen first and may use the configured stronger Cloud model only as a structured interpretation fallback. General answer synthesis is no longer allowed to handle probable controls using unrelated evidence such as a room list.
